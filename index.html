BenchScraper

Compare your costs in one place!

About

Don't have time to price compare at multiple stores before you order stock?

BenchScraper provides a simple interface for you to see current prices across major retailers and distributors, as well as a RESTful api for developers looking to integrate pricing into their apps.

API

The BenchScraper API is a RESTful API to provide food pricing info scraped from various places on the web.

With a super simple interface, it's easy to get started querying!

    Endpoints

  • GET /api/v1/items
  • View all scraped items, item attributes are name, price & seller
  • GET /api/v1/items?attribute=value
  • Search for a specific item based on name, price or seller attributes

    Setup

  • Clone the repo from GitHub
  • Install dependencies:
  • npm install

  • Seed data:
  • npm install -g node-mongo-seeds
  • seed-setup then modify the generated JSON file to match your db location
  • mongod in a different shell instance
  • NODE_ENV=*your environment* seed

  • Start the Server:
  • MONGODB_URI=*your mongodb location* npm start And query away!

    Tests

  • Run all tests with npm test

    Stack

  • Node.js & Express
  • Mongodb & mongoose ORM
